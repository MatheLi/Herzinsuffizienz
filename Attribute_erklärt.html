Erklärung der Attribute

Attribut Erklärung
Alter Im Alter steigt das Risiko für eine Herzschwäche
Geschlecht Geschlecht des Patienten (Frauen sind häufiger betroffen)
Brustschmerz Art Unterschiedliche Typen von Brustschmerzen
Blutdruck im Ruhezustand Ein abnormaler Blutdruck kann Anzeichen für eine Herzschwäche sein
Puls unter körperlicher Last Kriterium für die Herzleistung, häufig wird die Pulsfrequenz zur Kompensation einer eingeschränkten Kammerpumpleistung erhöht.
Blutzucker vor dem Essen Blutzucker gibt nähere Angaben über den Gesundheitszustand, Hinweis auf einen Diabetes mellitus als Risikoparameter für eine Herzinsuffizienz
Cholesterin-Spiegel Ein hoher Cholesterin-Spiegel kann zu koronaren Herzerkrankung führen, die sehr häufig Grund für eine Herzinsuffizienz ist.
Luftnot oder Angina Pectoris Lässt auf eine akute Herzinsuffizienz schließen → dringende Kontaktsuche mit einem Arzt
ST-Depression bei körperlicher Betätigung im Verhältnis zur Ruhezeit Auffälligkeiten im EKG
Steigung des ST-Segments Kann Hinweis auf eine akute Gefäßverengung geben
